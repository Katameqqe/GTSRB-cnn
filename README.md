# OCR_SIGN

This project provides an Optical Character Recognition (OCR) pipeline for extracting and analyzing signatures from images or scanned documents.

## Features
- Detects and extracts signatures from images
- Performs OCR to recognize text
- Visualizes results and provides output data

## Requirements
Install the required dependencies:

```bash
pip install -r requirements.txt
```

## Usage
Run the main notebook to process your images:

```bash
jupyter notebook main.ipynb
```

Follow the instructions in the notebook to upload images and view results.

## File Structure
- `main.ipynb`: Main notebook for running the OCR and signature extraction pipeline
- `README.md`: Project documentation
- `requirements.txt`: Python dependencies

## Example
Example usage and results are shown in `main.ipynb`.
